🐾 Agropet Shop Machado

 🚀 Tecnologias Utilizadas

 - React.js
 - JavaScript
- HTML5
- CSS3
- Vite

📸 Funcionalidades

- Navegação com menu responsivo
  
- Seções de serviços de banho, tosa e produtos
  
- Cards de serviços com imagens e preços

- Estilização moderna e responsiva
